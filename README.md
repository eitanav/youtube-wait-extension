# 🎬 YouTube Wait Extension

תוסף Chrome שמאלץ אותך לעצור, לחשוב, ולבחור במודע - לפני שאתה נכנס ליוטיוב.

![logo](icons/icon128.png)

## למה?

יוטיוב נבנה כדי לתפוס אותך. רוב הכניסות לאתר הן רפלקסיביות, לא מודעות. התוסף הזה מוסיף שכבת חיכוך פסיכולוגית כדי שכל ביקור יהיה החלטה אמיתית.

## פיצ'רים

- **🎯 בחירת כוונה** - "למה אתה נכנס?" עם 8 קטגוריות, כולל "סתם, לבזבז זמן" (Implementation Intentions)
- **⏱️ ספירת idle** - 10 שניות שמתאפסות בכל תזוזת עכבר/מקלדת. מאלץ ישיבה שקטה
- **📊 מעקב זמן** - כמה זמן בזבזת היום/השבוע ביוטיוב, מוצג באדום זוהר (Loss Aversion)
- **🎭 בחירה סופית** - "כן, קח את היום שלי" או "לא, יש לי חיים" עם פאן מתחלף
- **🏃 כפתור יציאה נסתר** - קטן, שקוף, וקופץ למיקום אקראי - צריך לרדוף אחריו
- **🕐 Grace Period** - אחרי שאישרת, חלון של 30 דק' בו ניווט נוסף פותח רק שאלה מהירה
- **⚙️ פאנל הגדרות** - 3 פריסטים (חופש/רגיל/לימודים) + טוגל לכל פיצ'ר בנפרד

## התקנה

### מ-GitHub (Load unpacked)

1. הורד את הקוד:
   ```bash
   git clone https://github.com/YOUR_USERNAME/youtube-wait-extension.git
   ```
2. פתח `chrome://extensions` בכרום
3. הפעל **Developer mode** (פינה ימנית עליונה)
4. לחץ **Load unpacked** ובחר את התיקייה

### דרישות מערכת
- Google Chrome / Edge / Brave (Chromium-based)
- Manifest V3 תמיכה (Chrome 88+)

## פיתוח

הקוד טהור - HTML/CSS/JS, ללא בנייה. סקריפטים עזר ב-Python (אופציונליים):

```bash
# יצירת אייקונים מחדש
python make_icons.py

# יצירת מפתח קבוע (כבר רץ פעם אחת)
python generate_key.py
```

## מבנה הפרויקט

```
youtube-wait-extension/
├── manifest.json       # הגדרות התוסף
├── background.js       # Service worker - יירוט ניווט + מעקב זמן
├── wait.html / .js     # מסך ההמתנה (קטגוריה → ספירה → בחירה)
├── popup.html / .js    # פאנל הגדרות (לחיצה על אייקון)
├── icons/              # אייקוני התוסף (16/32/48/128 px)
├── make_icons.py       # סקריפט ליצירת אייקונים
└── generate_key.py     # סקריפט ליצירת מפתח extension
```
